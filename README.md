# 🤖 Rule-Based Chatbot (Java)

A simple Rule-Based Chatbot built using Java that responds to user input based on predefined rules and keyword matching.

This project demonstrates fundamental concepts of Natural Language Processing (basic level), conditional logic, and interactive console applications using Java.

---

## 🚀 Project Overview

This chatbot interacts with users in a command-line interface and provides responses based on keyword detection.

It can:
- Respond to greetings
- Tell the current system time
- Introduce itself
- Provide help instructions
- Exit gracefully

The chatbot uses simple rule-based logic implemented with `if-else` conditions.

---

## 🎯 Features

- 🗣 Responds to greetings (hello, hi, hey)
- 👤 Introduces itself when asked
- ⏰ Displays current system time
- ❓ Provides help instructions
- 👋 Graceful exit using "bye" or "exit"
- 🔁 Continuous interaction loop

---

## 🛠 Technologies Used

- Java
- Scanner Class
- LocalTime (java.time package)
- Conditional Statements (if-else)
- String Processing
- Looping (while loop)
